Bonjour et bienvenue dans notre jeu cree par des etudiants en "Licence 2 Informatique 2022-2023".
Lisez attentivement ce fichier pour mieux comprendre le fonctionnement de notre jeu et pour vivre une meilleur experience. 
1-le but de ce jeu est: 
    -le joueur doit ranger des caisses sur des cases cibles.
    -Il peut se deplacer dans les quatre directions, et pousser (mais pas tirer).
    -Une fois toutes les caisses rangees (c'est parfois un vrai casse-tete), le niveau est reussi et le joueur passe au niveau suivant,
      plus difficile en general.
    -L'ideal est de reussir avec le moins de coups possibles (deplacements et poussees).
2-INSTRUCTION DU FONCTIONNEMENT DU JEU:
   -Apres avoir lu ce fichier nous allons vous demander de choisir entre une version graphique ou textuelle (je vous conseille la verssion graphique).
   -les deux versions ont des niveaux simillaires.
   -2.1-VERSION GRAPHIQUE:
    -Une fois la fenetre ouverte vous devez appuyer sur "ESPACE" pour lancer le Jeu.
    -Le niveau ou vous vous etes arretes, sera alors lance.(Niveu 1 si vous lancez le jeu pour la premiere fois).
    -Utilisez les fleches du clavier pour bouger le joueur.
    -la combinaison "Ctrl+Z":pour le retour en arrière.
    -la combinaison "Ctrl+R":pour recommancer le niveau.
    -Une le niveau termine appuyez sur "ENTREE" pour passer au prochain niveau.
    -la touche "echap" pour acceder au menu du jeu.
    -le "clique gauche" de la souris sur une case du jeu, vous permettra d'avoir le plus court chemin vers cette case(s'il éxiste).
   -2.2-VERSION TEXTUELLE:  
    -Legende :
      # : mur
      $ : caisse
      . : cible caisse
      , : cible joueur
      * : sous-monde
      @ : joueur
      + : joueur sur cible joueur ou caisse sur cible caisse zone de rangement 
      - : joueur sur cible caisse ou caisse sur cible joueur
    -Mouvement :
      s : bouger vers le bas
      q : bouger vers la gauche
      d : bouger vers la droite
      z : bouger vers le haut
      a : retour en arriere
      r : recommancer le niveau
3- A propos:
    - Les niveaux 20 & 21 represente la version "I'm World" dans notre jeu.
    - Le joueur peut absorber une caisse( plusieurs caisses), a vous de trouvez comment pour valider le niveau.
